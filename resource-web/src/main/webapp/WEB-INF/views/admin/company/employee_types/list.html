Employee type settings

Employees are grouped into Employee types (learn more)

You can set up any number of employee types, for example:

  1. Full day full-time employees (set with 8 working hours per day).
  2. Short day full time employees (set with 6 working hours per day).
  3. Flex time employees

When you add employees to Company, you will set their employee type.
